Guidelines followed while publishing a package in yantrashala.

  1. Create a neat readMe.
  2. Run code coverage and also integrate the module with codecov.io
  3. Integrate the module with travis.io for generating builds for every commit.
  4. Make your your code is clean as you are publishing to the world.
  5. Add an MIT License.
